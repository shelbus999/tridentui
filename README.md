## 🚀 Key Features

*   **Visual Drag-and-Drop**: Build layouts intuitively with a real-time canvas preview.
*   **Roblox Explorer**: Hierarchical object management with nesting and parenting logic identical to Roblox.
*   **Full Property Suite**: Fine-tune colors, fonts, Scale/Offset positioning, and modern UI effects (UICorner, UIStroke, UIGradient).
*   **Instant Luau Export**: Generate production-ready `.lua` code or download it directly to import into your game.
*   **Studio Accuracy**: Supports complex layouts (UIListLayout, UIGridLayout) and constraints.


---


### How to use

1. Open the builder.
2. Select an element from the left toolbar to add it to your screen.
3. Use the **Explorer** to organize your hierarchy.
4. Modify properties like **Size**, **Position**, and **Color** in the properties panel.
5. Click **Export Code** to get your ready-to-use Luau script!
